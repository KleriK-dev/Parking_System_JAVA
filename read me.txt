****BEFORE USING THE APPLICATION****

1. Copy and paste parkingsystem folder to C:\Users\[User]\Documents\NetBeansProjects\ParkingSystem\src

2. Go to phpMyAdmin and create a database and name it parking_system.

3. Then import the sql file ( parking_system.sql ).

4. Next create a user with the following data:
	
  •	User name: parkingSystem
  •	Host name: localhost
  •	Password: 1234

5. Add Data privileges (SELECT, INSERT, UPDATE, DELETE) on parking_system database for the user you just created.

6. Then open the application and log in as an administrator:

	Username: admin
	Password: admin
